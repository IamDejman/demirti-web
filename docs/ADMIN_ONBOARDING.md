# CVERSE LMS Admin Onboarding

## Access
- Go to `/admin` and log in with your admin credentials.
- If you need access, ask the system owner to add your admin account.

## Core workflows
1. Cohorts
   - Create cohorts in the **Cohorts** section.
   - Assign facilitators and enroll students.
2. Announcements
   - Create scheduled or immediate announcements.
   - Use **Publish now** to send immediately.
3. Jobs, Professionals, Projects
   - Curate job postings, industry professionals, and sample projects from the admin pages.
4. Certificates
   - Issue certificates by student email, then share verification link.
5. Moderation
   - Review reported messages in **Moderation** and resolve them.
6. Analytics + Exports
   - Review LMS metrics in **Analytics**.
   - Export LMS data from **Exports**.
7. Notification templates
   - Configure template defaults in **Notif Templates**.

## Operational tips
- Keep cohorts aligned with tracks and dates.
- Use announcements for key deadlines and live class updates.
- Check audit logs for critical changes.

## Support & troubleshooting
- Use **Impersonate** to open a learner session when diagnosing issues.
- Check `/api/health` to verify system health.
